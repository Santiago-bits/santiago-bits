### 🚀 Proyectos Destacados

📂 **Snake Online**  
Juego multijugador en red con una arquitectura cliente-servidor. Desarrollé la lógica del servidor en **C++** y el cliente interactivo en **JavaScript**, implementando patrones `async/await` para gestionar de forma eficiente el tráfico de red y evitar bloqueos en el servidor.

📂 **Plaxmar - Admin Panel**  
Panel de administración e-commerce full-stack desarrollado con **PHP** y **Bootstrap**. Diseñado para la gestión optimizada de productos, stock y categorías mediante el uso de bases de datos relacionales robustas.

📂 **Maze Game**  
Juego de laberinto web interactivo enfocado al 100% en la experiencia de usuario móvil. Implementa controles virtuales fluidos y un diseño completamente responsivo que se adapta a cualquier resolución de pantalla.
